# characterchooser

:rocket: Let the _FORCE_ choose your Star Wars identity.


## Install
```r
library(remotes)

install_github("MPCA-data/characterchooser")

```

## Use

Let the FORCE choose for you...
```r
library(haracterchooser)

charcterchooser()
```
>  1] ""


Choose yourself...
```r
charcterchooser("Yoda")
```
> [1] ""


```r
?charcterchooser
```
